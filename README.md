# armadillo

Armadillo is a modern password manager UI prototype built with React + TypeScript + Vite.

## Current state
- 3-pane desktop workspace (taxonomy, item index, detail/editor)
- Progressive mobile pane flow
- Credential detail editing (URLs, notes, security questions)
- Password generator controls
- Security center panel with risk states
- Armadillo-inspired visual design language

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
