# GDPR Workflow Notes (Draft)

## Scope
This app stores encrypted vault snapshots in sync backends. Decryption is client-side.

## Operational workflows
1. Tenant metadata export request
2. Tenant metadata deletion request
3. Retention policy review and log retention verification
4. DPA package preparation (subprocessors, security controls, transfer posture)

## Evidence expectations
- Request ticket
- Operator action log
- Completion timestamp
- Customer confirmation
